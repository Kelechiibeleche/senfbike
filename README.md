# SenfBike

SenfBike is a lightweight bike-sharing landing page prototype built for the city of Senftenberg, Germany.  
The project demonstrates how a small fleet bike-sharing service could be presented through a modern web interface focused on students, tourists, and local residents.

The goal is to provide a simple and intuitive experience where users can discover the service, understand how it works, and download the mobile app to start riding.

---

## Project Vision

SenfBike is designed as a digital mobility concept for smaller university towns where traditional large-scale bike-sharing systems are often too expensive or complex to implement.

The platform focuses on:

- affordable mobility
- sustainable transport
- student-friendly access
- simple app-based rentals

The idea is to start with a small pilot fleet and scale gradually if adoption grows.
